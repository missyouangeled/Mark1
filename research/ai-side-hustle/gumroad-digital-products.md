## 一、Gumroad 是什么？

Gumroad 是一个美国的数字产品电商平台，2011 年成立。**不用写代码、不用搭网站**，注册就能卖数字产品。自带市集（Discover）可以帮你引流。

## 三、注册 & 开店流程

### Step 1：注册账号

1. 打开 https://gumroad.com
2. 点「Start Selling」
3. 用 Google 账号登录（你已有）
4. 填用户名 → 你的商店地址就是 `https://你的用户名.gumroad.com`

### Step 2：设置收款（最关键一步）

进入 Settings → Payments：

| 收款方式 | 现状（2026年） | 适合你吗 |
|----------|-------------|:--:|
| **PayPal** | ✅ 2025 年初已恢复，可用 | ✅ 首选 |
| **银行转账** | 支持台湾银行 TWD，大陆银行需验证 | ⚠️ 待确认 |
| **Stripe** | 需香港/美国公司主体 | ❌ |

**推荐路径**：PayPal 收款。

### PayPal 中国账户注册

1. 打开 https://paypal.com/c2
2. 用你的身份证、国内手机号注册
3. 绑定国内银行借记卡（用于提现）
4. PayPal 中国 → Gumroad 后台授权关联

> ⚠️ **PayPal 中国账户的限制**：
> - 可以收款，可以提现到国内银行卡
> - 提现到国内银行卡有每年 $5 万美元的外汇额度限制
> - 每次提现 PayPal 收 $35 手续费（大额提现比较划算）

## 四、上架一个产品

### 产品信息填写

| 字段 | 怎么填 |
|------|--------|
| Name | 产品名称，英文为主 |
| Description | 详细描述，能用 AI 写，Markdown 格式 |
| Price | 美元定价，比如 $9.99 |
| Category | 选最贴近的 |
| Thumbnail | 封面图，建议 1000×1000 |
| File | 上传产品文件（PDF/Notion 链接/代码压缩包等）|

### 定价建议

| 产品类型 | 建议价格 |
|----------|---------|
| 单页 Notion 模板 | $3-9 |
| 模板套装（10+模板）| $15-29 |
| Prompt 合集（50+条）| $5-15 |
| 代码模板/脚本 | $9-49 |
| 工具 License（月费）| $4.99-19.99/月 |
| 教程/电子书 | $9-59 |

## 五、费用一览

```
你卖出一件 $10 的产品：

$10.00  产品售价
-$1.00  平台费（10%）
-$0.29  信用卡手续费（2.9%）
-$0.30  信用卡固定费
────────
$8.41   到你账户
```

| 费用项 | 比例 |
|--------|------|
| Gumroad 平台费 | 10% |
| 支付处理费 | 2.9% + $0.30 |
| 提现到 PayPal | 免费 |
| PayPal 提现到大陆卡 | $35/次 |

### 你还需要知道

- Gumroad 是 **Merchant of Record（MoR）**：平台代扣全球 VAT/GST/销售税，你完全不用管税
- 两周拨款一次，最低 $10 起拨
- 首次销售额未达 $100 时，邮件群发限制 100 人

## 六、怎么让人找到你的产品？

### Gumroad Discover 市集

你的产品会自动上架到 Gumroad Discover 内部市集。Gumroad 会根据搜索和浏览推荐你的产品。

### 外部引流

| 渠道 | 做法 |
|------|------|
| Reddit | 在 r/Notion、r/SaaS 等子版分享你的模板 |
| Medium/Blog | 写教程，文末链接到 Gumroad |
| Twitter (X) | 发布产品截图和使用演示 |
| YouTube | 录屏演示产品功能 |
| Product Hunt | 新品上架日发 |

## 七、替代平台一览

如果 Gumroad 的 10% 让你觉得高：

| 平台 | 费率 | 收款 | 适合 |
|------|------|------|------|
| Lemon Squeezy | 5%+$0.50 | PayPal→大陆 | ⭐ 低费率 |
| Polar | 5%+$0.40 | PayPal | 更便宜的 MoR |
| Payhip | 5% | PayPal/Stripe | 简单实用 |
| Ko-fi | 0-5% | PayPal | 小金额/打赏 |

> 💡 **建议**：可以先上 Gumroad（生态最大、流量最好），等卖起来后再同步上架 Lemon Squeezy/Polar 降低成本。
