## 一、按「产品类型」分——你做出东西，让人/市场买

### ① Chrome 浏览器插件
| 条目 | 数据 |
|------|------|
| 开发时间 | AI 辅助 2-8 小时一个 MVP |
| 上架成本 | $5 一次性注册费 |
| 变现方式 | 免费增值（$1-5/月）、买断 $2.99-9.99、广告 |
| 真实收入 | **绝大多数月入 $0-100**；头部 $1000-15000/月 |
| 成功案例 | ChatGPT Writer：70 万用户，月入 $15000（一个独立开发者） |
| 适合你吗 | ✅ 时间投入极少，做完放那不管 |

### ② Micro SaaS / 微型 Web 工具
| 条目 | 数据 |
|------|------|
| 是什么 | 一个解决特定小问题的在线工具，比如简历生成器、番茄钟、JSON 格式化 |
| 开发时间 | AI 辅助 3-7 天 MVP |
| 变现 | 订阅 $5-19/月 或 一次性 $9-49 |
| 真实收入 | 单人开发者常见 $1K-10K MRR；Micro SaaS 市场从 $15.7B 增长到 $59.6B |
| 成功案例 | 一个叫"Your Average Tech Bro"的 YouTuber：上班族，做了 14 个 app 后终于有一个月入 $7K-8K |
| 适合你吗 | ✅ 可以做，但需要持续维护和一定的营销投入 |

### ③ Telegram / Discord Bot
| 条目 | 数据 |
|------|------|
| 是什么 | 给 Telegram/Discord 群组用的自动化机器人 |
| 开发时间 | AI 辅助半天到 2 天 |
| 变现 | 按项目报价 $70-380/个（Fiverr 数据）；或 SaaS 订阅 $5-19/月 |
| 需求 | Freelancer 平台上有大量 Telegram bot 开发需求（加密分析、支付、通知推送等） |
| 适合你吗 | ✅ 开发快、需求明确、可以批量接 |

### ④ 数字产品（Gumroad / Etsy）
| 条目 | 数据 |
|------|------|
| 卖什么 | Notion 模板、代码模板、Prompt 合集、设计资源、AI 工具 License |
| 平台 | Gumroad（抽 10%）、Etsy、自建站 |
| 真实数据 | Gumroad 上 Software Development 品类总收入 $65.8M，单品均收入 $60,814（但方差极大） |
| 成功案例 | 掘金上一个程序员卖 Storybound（AI 视频自动化工具）License：月卡/年卡/永久，GMV 2 万 |
| 适合你吗 | ✅ 一次性制作、反复销售，真正的被动收入 |

### ⑤ 桌面软件 License（Windows/Mac）
| 条目 | 数据 |
|------|------|
| 是什么 | 打包成客户端的 AI 工具，卖激活码 |
| 案例 | 掘金博主嘟嘟 MD：用 Claude Code 开发 Storybound（文案→配图→配音→视频），卖月卡/年卡/永久 |
| 收入 | 他一个月 GMV 2W+（含视频号佣金），软件 License 是重要来源 |

## 二、按「服务类型」分——你帮别人做东西，按单收钱

### ⑥ Fiverr / Upwork 接海外单
| 条目 | 数据 |
|------|------|
| 做什么 | Chrome 插件、Discord/Telegram bot、API 集成、脚本开发 |
| 新手收入 | 前 3-6 个月 **$50-500/月** |
| 平台抽成 | Fiverr 20%，Upwork 10-20% |
| 冷启动 | 0 评价几乎等于没人理，前几单必须亏本做 |
| 当前趋势 | Fiverr 2026 年宣布转型高端市场，低端单在萎缩；AI 开发需求涨了 45% |
| 适合你吗 | ✅ 适合，但冷启动痛苦 |

### ⑦ 国内接单平台
| 平台 | 抽成 | 典型项目价格 | 状态 |
|------|------|------------|------|
| 程序员客栈 | 10-15% | 5K-15K | 需要审核入驻 |
| 猪八戒 | 5-20% | 3K-8K | 传统，竞争激烈 |
| 电鸭社区 | 0% | 5K-10K | 远程兼职，社区型 |

> ⚠️ **重要警告**：一篇 36 氪深度报道指出——AI 把小程序/APP 定制开发价格从 20 万打到了 2 万。定制开发的黄金时代已经结束了。不要走"接传统外包"这条路。

| 条目 | 数据 |
|------|------|

| 单价 | 5K-50K 元/个 |

## 五、几个你必须知道的残酷真相

### 来自 36 氪的真实报道：
> 一个做了 12 年的广州程序员，看着小程序报价从 12 万跌到 2 万。客户直接掏出手机展示 AI 生成的框架说"AI 都能免费做，你怎么还这么贵？"——最终 1.8 万成交，做了 17 天，身心俱疲。他的原话："AI 的入侵加上行业内卷，IT 工程师降薪差不多 35%。"

**结论：传统接外包的路已经被 AI 砸穿了，不要走。**

### 来自 InfoQ 的建议：
> "2026 年还想靠接单赚钱，别靠体力，要靠壁垒。"

**结论：你不该去跟印度/巴基斯坦开发者卷价格。你该做的是产品——做完一次，反复卖。**

### 来自掘金博主的血泪教训：
> "我去年的 ComfyUI 赛道，今年不做了就直接没收入。这不是真正的复利赛道。"

**结论：做内容的本质是卖时间，停了就没收入。产品和数字资产才叫被动收入。**

## 六、所以，你的最优策略是什么？

基于你的条件（有正职、时间有限、有编程基础、有我这个 AI 副驾驶），你最应该做的三类事：

### 🥇 第一梯队：做产品（一次投入，长期收益）
- **Chrome 插件**（最快出成果，开发时间 1-3 天）
- **数字产品/模板**（Notion 模板、代码模板、Prompt 合集）
- **Telegram/Discord Bot**（需求明确、开发快）

### 🥈 第二梯队：Micro SaaS
- 一个解决特定痛点的小 Web 工具
- 比插件重一点，但收入天花板高很多

### 🥉 第三梯队：Fiverr 标准化服务
- 不接"帮我做个 APP"这种无底洞
- 只提供标准化服务："我帮你做一个 Chrome 插件，3 天交付"
