## 二、路线 ①：上架 Coze（扣子）模板市场

### Coze 是什么？

字节跳动旗下的零代码 AI Agent 搭建平台。你通过拖拽搭出智能体，然后发布到模板市场。

### 两个版本

| | 国内版 (coze.cn) | 国际版 (coze.com) |
|---|:--:|:--:|
| 语言 | 中文 | 英文 |
| 模型 | 豆包大模型（免费额度） | GPT-4o/Claude（Credits 点数制） |
| 市场 | 扣子模板市场 | 海外 |
| 收费 | 免费版够用 → 进阶版按月 | 免费 10 Credit/天 |
| 适合你 | ✅ 先走国内版 | 后续拓展海外 |

### 国内版费用

| 档位 | 适合 | 价格 |
|------|------|------|
| 个人免费版 | 开发测试 | ¥0 |
| 个人进阶版 | 高频使用 | 按月订阅 |
| 高阶/旗舰版 | 专业开发者 | 更高额度 |
| 企业版 | 团队协作 | 按席位 |

> 个人开发测试用免费版完全够。

### 怎么做一个能卖的 Agent？

#### 典型流程

1. 想一个场景（比如"AI 帮电商老板自动处理退货退款"）
2. 登录 coze.cn → 新建智能体
3. 配置三个东西：
   - **人设与指令**（System Prompt）：告诉 AI 它是干什么的
   - **工作流**（拖拽拼装）：输入 → 查订单 → 判断 → 回复
   - **知识库**：上传产品手册/FAQ 文档
4. 测试 → 发布到模板市场

#### 能做些什么？

| 场景 | 模板内容 | 参考定价 |
|------|---------|---------|
| 电商退货处理 | 自动查物流 → 判断退货条件 → 发退款链接 | ¥199-299 |
| 小红书文案 | 输入产品 → AI 生成多版本文案+配图方案 | ¥99-199 |
| 简历优化 | 输入原简历 → AI 分析+改写+格式优化 | ¥49-99 |
| 客服分流 | 用户问问题 → 自动分类 → 转人工或自动回复 | ¥199-499 |
| 每日 AI 简报 | 抓取新闻 → AI 摘要 → 推送到飞书/微信 | ¥99-199 |

#### 怎么收款？

你不在 Coze 平台内收款。Coze 目前**没有**内建的创作者收入分成系统。

你需要：
1. 模板发布到 Coze 市场（引流）
2. 在商品描述里引导到你的 Gumroad/LemonSqueezy 购买页
3. 用户在 Gumroad 付款后，你发给他们模板文件（工作流 YAML 导出文件）

> 参考案例：有开发者在知乎分享，做了一个电商订单自动化工作流模板，标价 ¥299 在 Coze 商店销售。

### 开发一个模板要多久？

用 AI 辅助（你发需求，我写 Prompt/工作流配置）：**半天到 2 天**

## 三、路线 ②：GPT Store（OpenAI 官方市场）

### 现状（2026年6月）

| 事项 | 状态 |
|------|------|
| 商店开放 | ✅ 已开放 |
| 上架要求 | ChatGPT Plus/Team 账号（$20/月） |
| 赚钱方式 | 分成计划「开发中」，目前仅**美国创作者**小范围测试 |
| 中国创作者 | ❌ 尚不能直接参与收入分成 |

### 那中国人怎么靠 GPT Store 赚钱？

关键洞察：**GPT Store 目前不是直接赚钱的地方，而是引流工具。**

#### 策略：GPT Store = 免费广告位

1. 做一个高质量的 Custom GPT（比如"简历优化 GPT"）
2. 在 GPT Store 免费发布
3. GPT 的指令里引导用户访问你的网站/Gumroad/LemonSqueezy
4. → 在外部平台卖高级版本/私人 GPT/咨询

#### 怎么做 Custom GPT？

1. 订阅 ChatGPT Plus（$20/月）
2. 左侧菜单 →「Explore GPTs」→「Create」
3. 填写：
   - Name、Description
   - Instructions（核心提示词）
   - Conversation starters（开屏问题）
   - Knowledge（上传文件做知识库）
   - Capabilities（是否需要联网/图片生成/代码解释器）
4. 发布 → 选择 Everyone

> 开发一个 Custom GPT 只需要 **1-2 小时**

## 四、路线 ③：接定制 Agent 外包（备选）

### 怎么接？

| 平台 | 做什么 | 价格 |
|------|--------|------|
| Fiverr | "I will build you a Telegram/Discord bot" | $70-380/单 |
| Upwork | "AI automation consultant" | $60-150/小时 |
| 程序员客栈 | 智能体系统定制 | ¥5K-15K/单 |
| 小红书/即刻发案例 | 老板来私信 | 自己谈价 |

### 接单注意事项

> ⚠️ 新浪财经 2025 年深度报道：
> - 国内 C 端用户对工具类智能体**付费意愿低**
> - B 端大企业订单被大厂/SV 服务商包揽，个人难进
> - 中小微企业预算有限，只能赚点小钱
> - **低代码+同质化严重**，同类 Agent 扎堆，价格暴跌

### 所以如果你要走这条路

只接**有明确痛点+有预算**的客户：
- 电商卖家（退货自动化、批量上架）
- 自媒体运营（内容生成工作流）
- 小型 SaaS 公司（客服机器人）

不要接：个人、学生、"先做出来看看效果"
