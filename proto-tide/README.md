# PulseNest Prototype

一个原创的 **TapTap 气质**前端原型站，但没有直接复刻品牌、素材或文案。

## 技术栈

- Next.js 15
- React 19
- Tailwind CSS 4
- TypeScript

## 页面

- `/` 首页
- `/discover` 发现页
- `/rankings` 排行榜页
- `/game/[slug]` 游戏详情页

## 本地运行

```bash
cd proto-tide
npm install
npm run dev
```

打开 <http://localhost:3000>

## 构建

```bash
npm run build
npm run start
```
