## 2026-05-25：贾维斯语音随身助手 — 蓝牙耳机实时对话方案

### 来源
用户设想：租云服务器 + 自制能上网的蓝牙耳机 → 语音对话 → 耳机直接播回复。

### 架构
```
蓝牙耳机(ESP32) ──POST音频──→  云服务器
                                ├─ Whisper STT (GPU)
                                ├─ 贾维斯 (API)
                                ├─ ChatTTS 合成 (GPU)
                                └─ 返回音频 ──→ 耳机播放
```

### 延迟优化

**串行链路（优化前）：20-40s**
```
说完 → 全转文字 → 全想完 → 全合成 → 播放
```

**流式接力（优化后）：2-6s**
```
边说边转 → 第一句就开想 → 边想边合 → 边合边播
```

| 手段 | 提速 | 说明 |
|---|---|---|
| 流式接力（STT→推理→TTS 并行） | 🔥🔥🔥 | 最大收益，不等排队 |
| 模型常驻显存 | 🔥 | 省加载时间 |
| 短回复走 Edge TTS | 🔥 | "嗯我在"秒回 |
| 高频回复预缓存 | ⚠️ | "我在呢"直接播 |

### 所需组件

| 组件 | 方案 | GPU |
|---|---|---|
| 语音识别 | Whisper (GPU) | RTX PRO 6000 |
| 语音合成 | ChatTTS (GPU) | RTX PRO 6000 |
| 推理 | DeepSeek API | 远程 |
| 耳机硬件 | ESP32 + I2S 麦克风 + 喇叭 | — |
| 云服务器 | 租用 | 待选 |
