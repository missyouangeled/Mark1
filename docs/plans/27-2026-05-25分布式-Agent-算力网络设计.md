## 2026-05-25：分布式 Agent 算力网络设计

### 来源
用户咨询如何组合手头三台机器 + 云 GPU，设计分布式 Agent 算力网络。

### 节点资产

| 节点 | CPU | RAM | GPU | 角色 |
|---|---|---|---|---|
| 公司 Linux (HUB) | 4 vCPU R7-5800X | 7.7G | ❌ | 总控 + 任务路由 |
| 公司台式机 | 未知 | ⚠️ 4G（坏一根条→修到8G） | GTX 1080 (8G) | 本地 GPU 算力 |
| ROG 掌机 | Z1 Extreme | 16G 共享 | RDNA3 iGPU | GUI Agent + 移动站 |
| 云 GPU | 按需 | 按需 | **RTX PRO 6000 (96G)** | 重型算力 ¥2-3/h |

### 架构
```
                ☁️ RTX PRO 6000 (96G)
                ├─ 大模型推理 (70B+)
                ├─ ChatTTS 满血 GPU
                ├─ FLUX Pro 图像
                ↕ SSH/API 按需唤醒

公司 Linux (HUB — 永远在线)
├─ Gateway · 浏览器 · 分发
├─ 监工 · 健康 · 语音
└─ 任务路由
    ↕ SSH           ↕ HTTP/MCP
🖥️ 台式机        🎮 ROG 掌机
 GTX 1080         Agent-S GUI
 本地 SD/LLM      可视化浏览器
 ChatTTS GPU      Windows 专有任务
```

### 节点分工

| 节点 | 擅长 | 典型任务 |
|---|---|---|
| HUB | 总控、稳定、在线 | 搜索、运维、分发、聊天 |
| 台式机 | 本地免费 GPU | SD/FLUX 图像、7-13B LLM、ChatTTS GPU |
| 掌机 | 有屏幕、Windows | Agent-S GUI、验证码、可视化、移动中转 |
| 云 GPU | 96G 显存碾压一切 | 70B+ 大模型、FLUX Pro、视频处理、训练 |

### 优先级
