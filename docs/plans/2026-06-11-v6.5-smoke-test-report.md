## 5. ⚠️ 已知次要问题

**infos-handle same-origin 路由问题**

- 现象：浏览器从 `http://127.0.0.1:18789` 加载页面后，branding JS 使用 same-origin 请求 `/v1/query/...` 端点
- 实际：请求打到 gateway (port 18789)，gateway 返回 `index.html` 而不是 JSON
- 影响：branding 代码 fallback 到 broker snapshot 静态文件，功能不受影响
- 需修复：改 `infosHandleBaseUrl` 指向 unified proxy (`http://127.0.0.1:18788`)，或在 gateway 上配置 API 路由代理

## 总结

**本次升级核心问题（let i 重声明 → Control UI 黑屏）已解决。**
