## gemini-web2api — 免费 Gemini API 代理（调研保存）

- **时间**：2026-06-12
- **来源**：https://github.com/Sophomoresty/gemini-web2api
- **原理**：模拟浏览器访问 gemini.google.com 网页版，包装成 OpenAI 兼容 API
- **特点**：零成本、单文件、支持 Flash/Thinking/Pro、~2万字输出、联网搜索、Function Calling
- **风险**：Google 账号有封号风险、匿名模式限模型、接口可能变化
- **判断**：适合临时免费大容量输出，不适合做日常工作主力
- **状态**：暂不部署，方案已保存
