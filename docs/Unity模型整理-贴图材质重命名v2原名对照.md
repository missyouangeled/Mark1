## 改名规则

| 变更 | 示例 |
|------|------|
| 后缀全称化 | `_Col`→`Color`, `_N`→`Normal`, `_H`→`Height`, `_diff`→`Diffuse` |
| Albedo/Diffuse/Color 不互转 | `_Albedo`保持, `_diff`→`Diffuse`, `_Col`→`Color` |
| Mask 统一 | 所有 Mask 变体→`Mask` |
| Displacement/Height | `_Displacement`/`_disp`/`_H`→`Height` |
| 倒装+复合词拆分 | `castle_wall_slates`→`Wall_Castle_Slate`, `BronzeOld`→`Bronze_Old` |
| 数字补零 | `new2`→`new_02`, `Glass1`→`Glass_01` |
| 清除供应商前缀 | `TCom_`/`TexturesCom_` 移除 |
| 清除容量标记 | `_1K`/`_2K`/`_4K`/`_512` 移除 |
| 材质球加前缀 | `Mat_` |
| # 分隔材质属性 | `#red`/`#Dark`/`#Light` 保持 |
