## 四、风险与缓解

| 风险 | 影响 | 缓解 |
|---|---|---|
| LLM 真调超时不稳 | 异步测试 flaky | `pytest-timeout=5` + mock 全部 |
| CLI subprocess 慢 | 整套测试时间暴涨 | 用 `cli_runner` fixture + 短超时 (5s) |
| Engine 状态污染 | 测试间相互干扰 | conftest 已 autouse 隔离 tmp_path, 复用 |
| Armor 路径太多 | 写不完 10 测试 | 优先 P1 组合 (高覆盖率/低工时比) |
| 旧测试在 conftest reload 后 stale | 类似 test_fallback 失败 | 改用 `mocker.patch.object(module, "X")` 拿新引用 |
