## 一、P1-3: LLM 单元测试 Mock（让 CI 必跑）

### 背景

当前 `scripts/mark42_modules/llm_text_compressor.py` 测试 6「有 key 才跑」：

```python

```

**问题**: CI 环境无 key 时跳过，**LLM 调用逻辑没在 CI 跑过**，prompt 构造 / 响应解析 / 错误处理实际是否正确**没保证**。

### 目标

- 用 mock 让 CI 必跑真实 LLM 流程
- 验证: prompt 构造、模型名、token 上限、超时控制、错误回退
- **CI 永远 0 跳过**（不允许因环境差异跳过测试）

### 改动文件

| 文件 | 改动 | 行数预估 |
|---|---|---|
| `scripts/mark42_modules/llm_text_compressor.py` | 测试 6 改为 mock 必跑 | +~50 行 |
| 新文件 `scripts/mark42_modules/test_llm_mock.py`（可选） | mock fixture 集中管理 | +~80 行 |

### 关键设计决策

1. **Mock 库选择**: `unittest.mock`（标准库，无新依赖）
2. **Mock 注入点**: `_call_llm()` 内部的 `urllib.request.urlopen()`（line 255）
3. **Mock 策略**: 用 `patch("urllib.request.urlopen", mock_urlopen)` 注入
4. **Fixture 准备**: 准备 3 类响应 fixture（正常 / 错误 / 超时）
5. **不引入新依赖**: 不加 `responses` 库（标准库足够）

```python
"""Mark42 压缩子系统性能基准"""

import json
import time
import tracemalloc
from pathlib import Path
from statistics import mean, median, quantiles

def gen_samples(size_kb: int, n: int = 100) -> list[str]:
    """生成 N 个 size_kb 大小的样本"""
    samples = []
    for i in range(n):
        # 中文 + 代码 + log 混合
        text = "总而言之，第 001 段长文本。\n" * (size_kb * 1024 // 22)
        text += "def foo(x):\n    return x * 2\n" * 100
        text += "2026-06-25 INFO test message\n" * 50
        samples.append(text[:size_kb * 1024])
    return samples

def bench_algo(algo_fn, samples: list[str]) -> dict:
    """测量算法的延迟/内存"""
    latencies = []
    peak_mems = []

    for s in samples:
        tracemalloc.start()
        t0 = time.perf_counter()
        try:
            algo_fn(s)
        except Exception:
            pass
        elapsed = time.perf_counter() - t0
        current, peak = tracemalloc.get_traced_memory()
        tracemalloc.stop()

        latencies.append(elapsed * 1000)  # ms
        peak_mems.append(peak / 1024)     # KB

    return {
        "p50_ms": median(latencies),
        "p95_ms": quantiles(latencies, n=20)[18],  # P95
        "p99_ms": quantiles(latencies, n=100)[98], # P99
        "peak_mem_kb_p50": median(peak_mems),
    }

def main():
    from text_compressor import text_compress
    from log_deduplicator import logdedup
    from code_compressor import codecrush
    from diff_compressor import diff_compress
    from compression_algorithms import smartcrush
    from llm_text_compressor import llm_text_compress

    algos = {
        "smartcrush": smartcrush,
        "text": text_compress,
        "code": codecrush,
        "log": logdedup,
        "diff": diff_compress,
        # "llm": llm_text_compress,  # 单独测, 不参与主表
    }

    results = {}
    for size in [1, 10, 100, 1024]:
        print(f"\n=== {size}KB samples ===")
        samples = gen_samples(size, n=20)  # 样本量, 调小点
        for name, fn in algos.items():
            r = bench_algo(fn, samples)
            results[f"{name}_{size}kb"] = r
            print(f"  {name:12s}: P50={r['p50_ms']:6.2f}ms P95={r['p95_ms']:6.2f}ms "
                  f"Mem={r['peak_mem_kb_p50']:6.0f}KB")

    # LLM 单独测
    if os.environ.get("LLM_API_KEY"):
        samples = gen_samples(10, n=5)
        llm_results = bench_algo(llm_text_compress, samples)
        results["llm_10kb"] = llm_results
        print(f"  llm          : P50={llm_results['p50_ms']:6.0f}ms "
              f"Mem={llm_results['peak_mem_kb_p50']:6.0f}KB")

    Path("docs/design/mark42-压缩方案-性能基准-202606XX.md").write_text(
        format_report(results)
    )

def format_report(results: dict) -> str:
    # ... Markdown 报告
    pass

if __name__ == "__main__":
    main()
```

```python
        queue = get_compress_queue()
        deadline = _time.time() + 30  # 等 30 秒

        result["compression_drained"] = queue.stats.get("queue_size", 0) == 0
```
