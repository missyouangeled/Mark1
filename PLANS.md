## 2026-04-29：云服务器部署方案

### 用途
让 OpenClaw（贾维斯）24小时在线，随时可访问。

### 推荐服务商
| 服务商 | 推荐套餐 | 参考价格 | 说明 |
|--------|---------|----------|------|
| 腾讯云 | 轻量应用服务器 2核2G | ≈30元/月 | 性价比高，微信登录方便 |
| 阿里云 | 轻量应用服务器 2核2G | ≈34元/月 | 最稳，国内节点多 |
| 华为云 | 弹性云服务器 2核2G | ≈35元/月 | 也不差 |

### 配置建议
- CPU：2核
- 内存：2GB
- 系统盘：40GB
- 带宽：4Mbps 起步
- 系统：Ubuntu 22.04 / 24.04

### 部署步骤（摘要）
1. 服务器装 Node.js 22.x
2. `npm install -g openclaw`
3. 从旧机导出 tar.gz：`bash scripts/export-setup.sh`
4. 将 tar.gz 传至服务器并解压还原
5. `git clone` workspace
6. `openclaw gateway start`
7. 配好开机自启

### 访问方式
- WebChat：浏览器打开服务器IP:端口
- 微信通道：服务器配好微信即可手机端聊天

### 每月成本
30–35元/月。
