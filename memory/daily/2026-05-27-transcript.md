### 时间线
- **2025 年 4 月 13 日**：GitHub 开始对大陆 IP 实施限制，未登录用户直接 403
- 之后时好时坏，时而能直连时而不能
- 最近（2026 年 5 月）越来越多的开发者反馈"登录困难、页面加载卡死、验证码刷不出"

### 为什么会这样
不是单一原因，而是几层叠加：

1. **DNS 污染**：运营商的 DNS 故意返回错误 IP，浏览器找不到真正的 GitHub 服务器
2. **CDN 资源被阻断**：GitHub 的 CSS/JS 等静态资源托管在 CDN 上，这些节点经常被干扰，导致页面框架加载失败、登录按钮点不了
3. **登录鉴权链路长**：GitHub 登录需要调用多个 API + 2FA 验证，中间任何一个环节丢包就会卡死或报 504
4. **没有正式宣布"封禁"**：和 Google/Facebook 那种明确封锁不一样，GitHub 更像是被持续干扰、间歇性限制
