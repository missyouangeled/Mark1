### 三层架构

```
voice-reply.py --text '...' --scene morning|tender|night|warm
    ▼
L1: VoiceOrchestrator（调度器）
    - 场景 → 语音参数 (tempo/pitch/style)
    - 关键词浅层微调 ("想你了" → 再降 5%)
    - 引擎选择 + 降级链路
    - 文本分块 + 批量合成 + 后处理
    ▼
L2: 引擎层 (Noiz > ChatTTS > EdgeTTS)
    ▼
L3: 后处理 (ffmpeg: tempo + pitch + trim + loudnorm)
```

